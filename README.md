Метод Pomodoro — це метод керування часом, розроблений Франческо Чірілло в кінці 1980-х. Цей метод використовує таймер для того, щоб розбити роботу на 25-хвилинні інтервали, які називаються «pomodori» (італійське слово, що означає «помідори») і розділені короткими перервами.

програма являє собою автоматизований термінальний помодоро таймер, який дозволяє вести статистику часу, що був проведений за роботою.

для зміни тривалості інтервалів роботи, перерви, та великої перерви потрібно модифікувати змінні work, break та lunch відповідно, у форматі " 'кількість хвилин' * 60 ".
таймер має наступний функціонал:
- можливість ставати на паузу посеред будь-якого проміжку часу (на клавішу enter)
- автоматична пауза між циклами роботи та перерви (продовжити після паузи теж на клавішу enter)
- зручний рахівник відпрацьованих робочих сесій
- можливість виходу з програми та перегляд кінцевих даних після виходу (q + enter)
- створення файла statistics.json у папці data в місці, в якому знаходиться файл коду програми. файл автоматично поповнюється даними про відпрацьований час за датою на вашому комп'ютері в момент виходу та завершення роботи

поточна версія програми не є кінцевою, і може бути покращена у майбутньому.
